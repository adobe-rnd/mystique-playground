# mystique-playground

## Configuration

.env file

```shell
AZURE_OPENAI_API_KEY=...
AZURE_OPENAI_ENDPOINT=https://openai-west-us-tsaplin.openai.azure.com/
```

## Usage

To start the application, run the following command:

```shell
start.sh <url>
```

## Build

To build the application, run the following command:

```shell
build.sh
```
